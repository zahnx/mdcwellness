# Velora Wellness Spa

Velora is a modern, responsive spa and wellness website designed to showcase beauty and relaxation services through a clean and elegant UI/UX experience. Built with performance, scalability, and SEO in mind, it reflects a real-world business website implementation.

---

## Live Demo
https://velorawellness.vercel.app/

---

## Overview

Velora Wellness Spa is a front-end web application that highlights spa services such as massages, facials, and wellness treatments. The project focuses on delivering a seamless and visually appealing experience for users across all devices.

**Note:** This project is still a work in progress and will continue to be improved with additional features and enhancements.

---

## Features

- Fully responsive design (mobile-first)
- Clean and modern UI/UX design
- Fast performance powered by Vite
- Smooth animations using Framer Motion
- SEO optimized (meta tags, Open Graph, Twitter cards)
- Optimized social media sharing previews
- Reusable and scalable component structure

---

## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **Animations:** Framer Motion
- **Icons:** Lucide React

---

## Installation & Setup

Clone the repository:

git clone https://github.com/your-username/velora-wellness.git
cd velora-wellness

Install dependencies:

npm install


Run development server:

npm run dev

Build for production:

npm run build

**## SEO & Meta Optimization**

This project includes:
- Meta descriptions and keywords
- Open Graph (OG) tags for Facebook, LinkedIn, and WhatsApp
- Twitter card integration
- Favicon and PWA manifest setup
